# tic-tac-toe
Tic Tac Toe game coded in ruby

## To play
In order to play you shold run

```
$ ruby ti_tac_toe.rb 
```

There you will have instructions for playing different Tic Tac Toe games. For the moment the only available option is a Two Players game.

## Developers

To install:

```
$ bundle
```

To run tests:

```
$ rspec spec/
```
